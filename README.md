# NASforBeginners

Plain-English home server setup guides. Built with Next.js 14, Tailwind CSS.

## Deploy

Import this repository into Vercel. No environment variables required.

## Dev

```bash
npm install
npm run dev
```

## Structure

- `app/` - Next.js App Router pages
- `app/guides/` - Individual guide pages
- `app/tools/` - Interactive tools (power calculator, build recommender)
